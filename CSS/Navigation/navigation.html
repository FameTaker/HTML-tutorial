To create a working URL Path we need to organize our html files in specific folders. Create folders like main page, images, blog pages and write directions like blog/march9

There are two ways to write url path
first one /blog/march9 starts searching from main page and finds the blog folder and march9 webpage.
second one ../blog/march9starts searcing from the current location and finds the blog folder.
