In HTML we need to differentiate between emphasized text and italics. <i> element is used for italics and <em> is used for emphesizing and significance. Reason for this is for further stylisation on CSS.

Another way to highlight text is to use <strong> element and <b> element. Use strong element to show importance and seriousness. Again the reason for this is to help stylisation on CSS
