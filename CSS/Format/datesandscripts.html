This section is about superscripts, subscripts, and small texts

Superscript element is used for the characters that are set under the text baseline. Mathematics and notes are often use superscrips.
<sub> element is used for superscripts. For example H20
<sup> element is used for subscripts. For example when you need to mention a footnote in text2

the <small> is highlight the small text. It is not used to adjust the text size, it is for highlighting.
