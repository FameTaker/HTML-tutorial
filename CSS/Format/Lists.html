This section is about Quotes

There are many different ways to of quotes in HTML. There is a blockquote option to quote a whole paragraph. Although there are no direct differences visible for blockquote on HTML. The cite element is to mark the author. The q element is used to make quotation marks on the web page. It is helpful to use in CSS styling.
