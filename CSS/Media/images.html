This section is about Images

Every image should have the 4 attributes. The img attribute, the source src attribute, the altexplenation attribute and width,height attribute.

The alt text should be simple and can be left blank
The width and height attributes are always mean pixels. The order does not matter. When not included the actual size of the picture is applied.
HTML can support many different image types however we should consider the pixel count and image size when deciding. JPG and PNG are default images but they can be heavy or break the page. SVG is the vectoral format. SVG is lightweight and can adjust itself to fit page however it is not an image but a code and

Responsive Images

In html different resolutions of the same image can be uploaded and then browser can decide which image to use.

we can use <picture> element for more configurations.
